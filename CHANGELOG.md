# Changelog

All notable changes to DresOS WebView are documented here.

## v1.0.0

First public release of DresOS WebView.

- System WebView provider `org.dresos.webview`, built from Cromite and rebranded for DresOS.
- Signed with the DresOS release key
  (SHA-256 `b7815f746c3183c66cad631079cb669c0494a8be542a2a96e5975d8fabcd92de`).
- Ships alongside the DresOS WebView Overlay Magisk module, which whitelists this
  provider in the framework.
- Internal build version: 145.0.7632.120 (versionCode 763212001).
